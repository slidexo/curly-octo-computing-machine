Privacy Policy for UserPhone

Last Updated:

This Privacy Policy explains how UserPhone ("the Bot") collects, uses, and protects your data when you use our cross-server voice and chat Discord bot. By using UserPhone, you consent to the practices described in this policy.

1. Data We Collect

UserPhone may collect and process the following data:

  • Server Information: Server IDs, channel IDs, and role IDs for functionality.
  • User Information: User IDs, usernames, and discriminators for cross-server communication.
  • Message Content: Temporary processing of messages for relay between servers (not stored long-term).
  • Voice Data: Metadata about voice channels (e.g., participant IDs) for voice chat features.

2. How We Use Data

The collected data is used solely for:

  • Enabling cross-server text and voice communication.
  • Providing moderation tools and server management.
  • Improving bot performance and fixing issues.

We do not sell or share your data with third parties.

3. Data Storage & Retention

  • Message Content: Not stored permanently; only processed in real-time for relaying.
  • Server/User IDs: Stored only as long as necessary for functionality.
  • Voice Data: No audio recordings are stored; only connection metadata.

4. Data Security

We implement security measures to protect your data, including:

  • Encrypted data transmission.
  • Restricted database access.
  • Regular security audits.

5. User Rights

You have the right to:

  • Request Data Deletion: Contact us to remove stored data associated with your server or user ID.
  • Opt Out: Remove the bot from your server at any time to stop data collection.

6. Changes to This Policy

We may update this Privacy Policy. Continued use of UserPhone constitutes acceptance of changes.
